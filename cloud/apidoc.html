API Documentation

Upload file (POST)

https://h1ax.site/cloud/api/upload.php


Request body: {file_contents}

  1. You will got response as a file id.
  2. Remember that: File upload on api cannot be downloaded from the official website and only can be download from api

Get File (GET)

https://h1ax.site/cloud/api/payload.php


Request body: id={your_file_id}

  1. You will got response as contents of the file you was upload.
  2. Then use it as your data.
